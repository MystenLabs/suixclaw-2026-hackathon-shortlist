## Problem & Solution
- **What is it?** A complete financial stack for AI agents on Sui — checking, savings, credit, exchange, and x402 API payments in one SDK.
- **What problem does it solve?** AI agents can hold and send tokens, but can't do anything smart with idle capital. No yield, no credit, no autonomous API payments. Every agent builder reinvents DeFi integrations from scratch.
- **Who has this problem?** Developers building autonomous agents that need to manage money and fund their own operations continuously.
- **Does the solution fit?** Yes — one npm install gives agents save/borrow/swap/repay/pay. The "agent pays its own rent via yield" thesis directly addresses agent sustainability.
- **Would someone use this?** Credibly yes — published on npm, works on mainnet, plugs into Claude Code/Codex/Copilot via Agent Skills.

## Description
"The first bank account for AI agents." One npm install gives any AI agent a complete financial stack — checking, savings (NAVI Protocol ~4-8% APY), credit (borrow against collateral), exchange (Cetus DEX), and x402 payments. All on Sui mainnet. Non-custodial, fully on-chain, verifiable.

## Code Review Notes
- **Repo structure:** Monorepo (Turborepo) with 4 packages: `contracts`, `sdk`, `cli`, `x402`
- **Lines of code:** ~7,986 lines of TypeScript across packages
- **Move contracts:** 7 source files + 2 test files (core, admin, treasury, constants, errors, events). Well-structured Move code with proper module separation, admin capabilities, fee configuration, and versioning.
- **SDK:** Full-featured — wallet management (Ed25519 keypair, AES-256-GCM encryption), NAVI Protocol integration (save/withdraw/borrow/repay), Cetus DEX swaps, gas management, yield tracking, sentinel monitoring. Uses `@mysten/sui` SDK directly.
- **CLI:** Clean command-line interface with init, send, save, borrow, swap, balance, pay commands.
- **x402:** Payment protocol implementation with facilitator, payment kit, client — includes test files with proper mocking.
- **Infrastructure:** Docker deployment configs, ALB setup, indexer — shows production-level thinking.
- **Agent Skills:** Separate `t2000-skills` directory with skill definitions for Claude Code, Codex, Copilot integration.

## Sui Integration Analysis
**Deep integration across multiple Sui stack components:**
- **Move smart contracts** — Config management, admin capabilities, treasury, fee structures deployed on mainnet
- **@mysten/sui SDK** — Direct SuiClient usage for transaction building, signing, querying
- **NAVI Protocol** — DeFi integration for savings/lending (supply, borrow, repay)
- **Cetus DEX** — On-chain token swaps with slippage enforcement
- **Gas Station** — Sponsored transactions for frictionless onboarding
- **PTB (Programmable Transaction Blocks)** — Atomic multi-step DeFi operations
- **x402 on Sui** — First implementation of the x402 payment protocol on Sui (Payment Kit with Move-level nonce enforcement)

## Overall Assessment
**Strong contender for Track 2.** t2000 is one of the most technically ambitious and complete projects in the hackathon. It's not a wrapper or a template — it's a genuine financial infrastructure layer for AI agents on Sui, with real code depth, real npm packages, and a real mainnet deployment. The Sui stack integration is the deepest I've seen across submissions. The x402 payment protocol implementation on Sui appears to be a first.

The main weakness is the demo — a 49-second video doesn't do justice to the scope of the project. On-chain activity is also thin (1 tx), which raises questions about real-world testing. But the codebase quality and architectural ambition are top-tier.

**Shortlist recommendation: Yes — strong candidate for top 5 in Track 2.**
