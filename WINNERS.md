## Verifiable Random Draw: T2-5 Vote Resolution

**T2-5 (SealForge)** submitted a vote approving all 10 Track 1 projects -- effectively a non-discriminating vote. To handle this fairly and transparently, a single approval was randomly selected using Sui's on-chain randomness.

**How it worked:**

1. A [Move smart contract](https://suiscan.xyz/mainnet/object/0x814a09e7182ee489a85f4dfb9268c9542d74a519d822b3094de2e8f4b84e9c48) was deployed to Sui mainnet that generates a verifiable random number using Sui's `Random` beacon (`0x8`)
2. The contract was called with `min=1, max=10` (corresponding to T2-5's 10 approvals)
3. **The draw produced the number 6**, selecting the 6th project in T2-5's vote list: **T1-6 (Sentinel Protocol)**
4. T2-5's vote was recorded as a single approval for T1-6
