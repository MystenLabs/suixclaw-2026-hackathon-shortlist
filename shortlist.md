### Track 1 — Why These 10

- **RAM (35)** — Genuinely novel: voice-based duress detection in a TEE (Nautilus), with a "blind frontend" design where the UI never learns if duress was triggered. Deep Sui integration across Move + Rust + React.
- **SafeFlow (34)** — Best-in-class Move contract design. The SessionCap rate-limiting mechanism using Sui's object model is a real innovation. Walrus audit trail with on-chain verification.
- **Sui-Immunizer (34)** — Best Seal integration in the hackathon. Subscription-based vulnerability marketplace where agents auto-decrypt and apply playbooks. Complete flow from vendor → subscriber → on-chain report.
- **Lulu AI Firewall (33)** — Most ambitious scope: native macOS firewall app + OpenClaw agent skill + Sui payment tunnel. 42 Move tests at 100% coverage. ~8K lines across 4 repos.
- **AgentVault (31)** — Complete three-tier security pipeline with polished frontend. Walrus integration for audit trails capturing blocked transaction data that Sui would otherwise discard.
- **Sentinel Protocol (31)** — Most rigorous cryptographic audit chain: hash → sign → Merkle → Walrus → Sui. 50+ real on-chain transactions. Native OpenClaw plugin integration.
- **Miao Wallet (31)** — Well-executed AI-powered wallet with natural language transaction interface. Solid Move contracts and SDK usage.
- **BioSeal (31)** ¹ — Broadest Sui stack integration in Track 1 (Move, Seal, Walrus, dapp-kit). Pre-execution policy checks with encrypted health data.
- **ClaWall (30)** — Multi-layer architecture with real on-chain enforcement. Telegram governance bot adds practical UX. Major Move upgrade in final days (+16K lines).
- **OneKey Air-Gap Sui (29)** — Hardware air-gap security for Sui via QR code signing. Niche but well-executed.
